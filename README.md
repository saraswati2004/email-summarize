# 📧 AI Email Summarizer

> **Transform cluttered inboxes into concise, actionable insights.**
> An AI-powered email assistant that securely fetches Gmail messages, intelligently categorizes them, filters sensitive content, and generates concise summaries using **Google Gemini**.

---

## ✨ Key Features

* 📥 Secure Gmail integration with **Gmail API**
* 🤖 AI-powered email summarization using **Gemini**
* 🗂️ Automatic email categorization
* 🔒 Sensitive email detection and filtering
* ⚡ Clean, modular, and scalable Python architecture

---

## 🛠️ Built With

| Technology           | Purpose                         |
| -------------------- | ------------------------------- |
| 🐍 Python            | Core application                |
| 📧 Gmail API         | Email retrieval                 |
| 🤖 Google Gemini API | AI summarization                |
| 🔐 OAuth 2.0         | Secure authentication           |
| 🌱 dotenv            | Environment variable management |
