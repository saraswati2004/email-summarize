AI Email Summarizer

An AI-powered application that fetches emails from Gmail, categorizes them, filters sensitive content, and generates concise summaries using the Gemini API.

Features
Fetch emails securely using the Gmail API
Generate AI-powered summaries with Gemini
Classify emails into categories
Skip sensitive emails
Simple and modular Python implementation

Tech Stack
Python
Gmail API
Google Gemini API
OAuth 2.0
dotenv
